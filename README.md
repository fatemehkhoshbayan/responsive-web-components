# Responsive Web Components

A small front-end project for building a **responsive** layout using plain HTML and CSS semantic page structure, mobile friendly viewport settings.

----

## Features

- **Semantic HTML5** — `header`, `main`, and `footer` for a clear document outline.
- **Responsive baseline** — viewport meta tag so layout scales correctly on phones and tablets.
- **CSS reset** — minimal global reset (`margin` / `padding`) for consistent cross-browser styling.

----

## Tech stack

| Area        | Choice                          |
| ----------- | ------------------------------- |
| Markup      | HTML5                           |
| Styles      | CSS3                            |
| Frameworks  | None                            |

----
